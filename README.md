# TheMovieDB
 This application allows a user to search for a movie by name and display the results in a tableview    
 The results will show a title, rating, summary and display the movie poster in a custom cell
This project excercised Closures, Asynchronous calls, URLSession, Failable initializers, Parsing through the returned data in JSON format from network APIs< Working with network API documentation, UITableView concepts such as custom cells.
